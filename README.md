## 📊 核心計算公式（完全透明）

```text
1. 課程總分鐘數
   T_Course = 已上課堂數 × 每堂課分鐘數

2. 允許最大缺席分鐘數（時間銀行上限）
   T_Max = T_Course × (1 − 最低出席率要求)

3. 已消耗分鐘數
   T_Consumed = (缺席堂數 × 每堂分鐘數) + 累計遲到分鐘數

4. 預期最終出席率（之後完美出席）
   R_Projected = (T_Course − T_Consumed) / T_Course × 100%

5. 剩餘緩衝時間
   Buffer = T_Max − T_Consumed
